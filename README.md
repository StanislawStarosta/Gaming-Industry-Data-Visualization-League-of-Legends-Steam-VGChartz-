# Gaming Industry Data Visualization Project

## 📌 Overview
This project focuses on data visualization and exploratory data analysis of the gaming industry using multiple datasets, including League of Legends statistics, Steam Charts, and VGChartz sales data.

The goal is to identify trends in:
- game popularity
- player base statistics
- revenue and sales performance
- demographic insights

---

## 📊 Tools & Technologies
- Python
- Pandas
- Matplotlib / Seaborn
- WordCloud
- Jupyter Notebook

---

## 📁 Project Structure
- `notebooks/` – Jupyter notebooks with analysis
- `data/` – raw datasets (CSV files)
- `images/` – generated plots and visualizations

---

## 📈 Key Analyses
- League of Legends player statistics and demographics
- Most popular games over time
- Steam player trends
- VGChartz sales and platform comparison
- Word cloud analysis of gaming data

---

## 🧠 Key Insights
- Significant differences in game popularity across platforms
- League of Legends maintains strong player engagement
- Revenue and player base do not always correlate directly
- Visual trends highlight dominance of specific franchises

---

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
```bash
pip install pandas matplotlib seaborn wordcloud
```
3. Open notebooks in Jupyter / VS Code
4. Run cells sequentially
